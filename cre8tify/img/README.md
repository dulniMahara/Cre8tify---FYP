# 📌 Cre8tify — “Wear Your Imaginations”

Cre8tify is a Sri Lankan-based web platform that enables users to browse, customize, and purchase custom-designed t-shirts, while allowing local designers to create, upload, and sell their artwork through a simple and user-friendly system.

## 🚀 Features
👕 Buyer Features

- Browse t-shirts for men, women, and kids
- View product details
- Live preview (user photo upload)
- Try on dummy 360° models
- Customize text, colors, minor edits
- Request designer edits
- Track requests & notification updates
- Pay and either download the design or purchase the product

🎨 Designer Features

- Upload and create new designs
- Track status (draft / submitted / approved / rejected)
- Accept & manage edit requests
- Send custom offers (price, time, notes)
- Track sales & earnings
- Receive notifications

🛠 Admin Features

- Manage accounts
- Approve/reject designs
- Manage reports
- Verify payments
- Adjust product prices and service fees
